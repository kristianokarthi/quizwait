# QuizWait 🧠

> The waitlist that rewards you for what you know — and who you bring.

**Live Demo:** Coming soon

---

## What is This?

A referral-powered waitlist app. Join, refer friends, climb the live queue. Top 50 at launch win 3 years of ChatGPT Plus or Claude Pro — free.

---

## Stack

- **Next.js 15** + TypeScript
- **Tailwind CSS**
- **Neon Serverless PostgreSQL**
- **Vercel**

---

## Local Setup

```bash
npm install
cp .env.example .env.local
# Add DATABASE_URL and NEXT_PUBLIC_APP_URL
npm run dev
```
